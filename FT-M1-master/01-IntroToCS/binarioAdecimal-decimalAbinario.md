De BINARIO a DECIMAL

Binario 101 a decimal:

Binario ------------>       1              0               1
Posiciones --------->       2              1               0
Calculo ------------>     (2^2)*1       (2^1)*0         (2^0)*1
-------------------->       4       +      0       +       1       =        5   (Binario 101 -> decimal 5)

---------------------------------------------------------------------------------------------------------->

Binario 110 a decimal:

Binario ------------>       1              1               0
Posiciones --------->       2              1               0
Calculo ------------>     (2^2)*1       (2^1)*1         (2^0)*0
-------------------->       4       +      2       +       0       =        6   (Binario 110 -> decimal 6)
